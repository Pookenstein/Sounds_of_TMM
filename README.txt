 Notes:
 - It is recommended testing at your current sound levels before playing. If it is too loud, you can either turn down your master volume in game, turn down your system volume, or change the WA to not load.
	- you may also check the sound level of each file in game by finding the audio file path in each weak aura by going to that WA's Action tab, under "Sound File Path", and pressing Enter.
	- If you have sound editing, you can also open it in a sound editor like Audacity and lower the Gain, and export it to the same name.
	- Any change to a sound file requires you to reload the game, in order to make the changes appear in game.
